# Claude Code Configuration

This directory contains configuration files and scripts for Claude Code.

## Key Features

- **Hooks**: TypeScript-based scripts that execute at Claude Code events (pre/post tool use, stop, etc.)
- **Command blocking**: Prevents dangerous or unwanted commands via ng-checker
- **Custom commands**: Markdown-based slash command definitions
- **Custom agents**: Specialized agent definitions
- **Custom skills**: Reusable skill definitions for common workflows
- **MCP integration**: Model Context Protocol server configuration
- **Status line**: Custom status line showing workspace and context info
